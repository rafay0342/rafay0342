## 🧠 [Rafaygen](https://rafaygen.cloud) — my flagship

A **multi-provider, agentic AI workspace** that picks the right model for every
prompt on its own — across **Groq, OpenAI, Qwen, Mistral, OpenRouter & local
Ollama** — then doesn't just answer, it **builds**:

- 🌐 Live web pages — HTML · Tailwind · React/JSX
- 📊 Office files — XLSX · DOCX · PPTX · PDF
- 📈 Charts · 🔷 Mermaid diagrams · 🎴 flashcards · ❓ quizzes · 🎨 SVG
- 💻 Runnable code in **8 languages**, executed in Docker sandboxes
- 🎙️ Voice in & out · 🖼️ image generation

> Designed, engineered, and shipped end-to-end — by one person.
