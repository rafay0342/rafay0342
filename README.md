## 🧠 Rafaygen — my flagship

> A **multi-provider, agentic AI workspace** that chooses the right model for
> every prompt on its own — then doesn't just answer, it **builds**.

It routes intelligently across **Groq · OpenAI · Qwen · Mistral · OpenRouter ·
local Ollama**, and turns a single chat box into a real production studio:

| | Capability |
|:--:|:--|
| 🌐 | **Web pages** — full HTML + Tailwind + React/JSX, rendered & editable live |
| 📊 | **Office files** — XLSX · DOCX · PPTX · PDF, generated server-side |
| 📈 | **Charts** — bar · line · pie · scatter, streaming-safe |
| 🔷 | **Diagrams** — Mermaid flowcharts, UML, sequence, state, C4 |
| 🎴 | **Flashcards** & ❓ **quizzes** — interactive, auto-scored |
| 💻 | **Code execution** — 8 languages in Docker-isolated sandboxes |
| 🎙️ | **Voice** — speech-to-text in, read-aloud out |
| 🖼️ | **Images** — on-demand generation |
